# Cara Menjalankan Project
## 1. Install Node js terlebih dahulu
https://nodejs.org/en

## 2. Download Ekstensi di vscodenya
- Prettier
- ESLint

## 3. Jalanin ini
```bash
npm install
npm run dev
```
Lalu buka ini [http://localhost:3000](http://localhost:3000) di browser
Kalau error berarti kurang bismillah
